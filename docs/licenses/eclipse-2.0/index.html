    Eclipse Public License 2.0

    لایسنس Eclipse Public License 2.0 یک لایسنس copyleft متوسط است که توسط بنیاد Eclipse ایجاد شده است. این لایسنس برای پروژه‌های نرم‌افزاری طراحی شده که نیاز به تعادل بین آزادی و کنترل دارند.

    ویژگی ها

    مجوزها

    • استفاده تجاری
    • توزیع
    • دستکاری
    • استفاده خصوصی
    • ثبت اثر
    • استفاده در نرم‌افزارهای اختصاصی (با شرایط)

    شروط

    • ثبت لایسنس و کپی‌رایت
    • ارائه سورس کد برای تغییرات
    • ثبت تغییرات
    • حفظ لایسنس برای تغییرات

    محدودیت ها

    • سلب مسئولیت
    • بدون گارانتی
    • تغییرات باید تحت EPL منتشر شوند

    تفاوت با GPL

    EPL نسبت به GPL انعطاف‌پذیرتر است و اجازه استفاده در نرم‌افزارهای اختصاصی را می‌دهد، اما تغییرات در خود نرم‌افزار باید تحت همین لایسنس منتشر شوند.
