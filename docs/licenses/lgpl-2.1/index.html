    GNU Lesser General Public License v2.1

    لایسنس GNU LGPL v2.1 یک لایسنس copyleft است که برای کتابخانه‌ها طراحی شده است. این لایسنس به شما اجازه می‌دهد تا از کتابخانه‌های تحت این لایسنس در نرم‌افزارهای اختصاصی استفاده کنید، اما تغییرات در خود کتابخانه باید تحت همین لایسنس منتشر شوند.

    تفاوت با LGPL v3

    LGPL v2.1 نسبت به LGPL v3 ساده‌تر است اما برخی ویژگی‌های مهم مانند حمایت از حق ثبت اختراع را ندارد. بسیاری از کتابخانه‌های مهم همچنان از LGPL v2.1 استفاده می‌کنند.

    کاربردها

    LGPL v2.1 معمولاً برای موارد زیر استفاده می‌شود: - کتابخانه‌های C و C++ - کتابخانه‌های سیستم - کتابخانه‌های که در پروژه‌های تجاری استفاده می‌شوند - کتابخانه‌های که نیاز به انعطاف دارند
